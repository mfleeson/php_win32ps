wb_set_range

bool wb_set_range (int control, int vmin, int vmax)

Sets the valid range of values (vmin and vmax) of a control. Valid classes are Gauge, ScrollBar, Slider and Spinner. Returns TRUE on success or FALSE if an error occurs.

NOTE: Due to limitations in Windows, values are limited to 16-bit values only.
